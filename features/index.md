# Detection Features

ScamShield uses five independent detection engines. Each runs on every message and can trigger its own set of actions and alerts.

## Detection Engines

| Engine | Detects |
| --- | :-- |
| Casino | Celebrity casino promos |
| Support | Masked/hidden URLs |
| IntroScan | Fake business pitch intros |
| QuickScam | Get-rich-quick schemes |
| ScamScan | Manual `/check` scans |

## Common Patterns - /check

ScamShield contains 15\+ pattern groups covering:

- Unsolicited DM Greetings
- Money Deposit / CashApp Scams
- False Report Scams
- Nitro Giveaway Scams
- Giveaway Scams
- Cryptocurrency Scams
- Fake Community Interest
- Support Impersonation
- Suspicious Links
- Urgency Pressure
- Payment / Account Requests
- Art Commission Scams
- Try My Game Scams
- Fake Introduction / Business Pitches
- Suspicious Keywords
