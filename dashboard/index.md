# Web Dashboard

The ScamShield web dashboard provides a graphical interface for managing all server settings. It communicates with the bot via IPC (Inter-Process Communication).

## Dashboard Tabs

| Tab | Purpose |
| --- | --- |
| Casino | Configure casino scam detection actions, alerts, and threshold |
| Support | Configure support scam, intro scan, and quick scam actions and alerts |
| General | Configure linked server propagation settings |
| Detections | View and delete detection history |
| Warnings | View and clear user warnings |

## Access

1. Navigate to the dashboard URL ([https://dashboard.scamshield.lol](https://dashboard.scamshield.lol))
2. Click **Login with Discord**
3. Authorize the application
4. Select your server from the list

Only server administrators can access and modify settings.
