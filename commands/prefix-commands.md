# Prefix Commands

Prefix commands use the `s.` prefix and can be used in server channels. They are text-based alternatives to slash commands.

## s.help

Display a list of all available commands grouped by category.

**Usage:** `s.help`

**Response:** A plain text message listing all slash commands organized by category.

## s.ping

Check the bot's connection latency.

**Usage:** `s.ping`

**Response:** `Pong! 42ms`
